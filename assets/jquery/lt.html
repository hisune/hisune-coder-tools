返回值:Array<Element(s)>:lt(index)

概述

匹配所有小于给定索引值的元素

参数

indexNumberV1.0

从 0 开始计数

示例

描述:

查找第一第二行，即索引值是0和1，也就是比2小

HTML 代码:
<table>
  <tr><td>Header 1</td></tr>
  <tr><td>Value 1</td></tr>
  <tr><td>Value 2</td></tr>
</table>
jQuery 代码:
$("tr:lt(2)")
结果:
[ <tr><td>Header 1</td></tr>, <tr><td>Value 1</td></tr> ]
