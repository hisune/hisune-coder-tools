返回值:jQuerysubmit([[data],fn])

概述

当提交表单时，会发生 submit 事件。

该事件只适用于表单元素。

参数

fnFunctionV1.0

在每一个匹配元素的submit事件中绑定的处理函数。

[data],fnString,FunctionV1.4.3

data:submit([Data], fn) 可传入data供函数fn处理。

fn:在每一个匹配元素的submit事件中绑定的处理函数。

示例

描述:

提交本页的第一个表单:

jQuery 代码:
$("form:first").submit();

描述:

如果你要阻止表单提交:

jQuery 代码:
$("form").submit( function () {
  return false;
} );
